# Declutter
epic program for my ocd brain

THIS ISNT VIBE CODED!!!!!!!!

## what it does
- shows installed packages that are explicitly installed, installed only as an optional dependency, and packages that are installed but arent needed.
- shows files and folders in the xdg dirs that dont match the name of an installed package.
- oh and it also just removes empty files and folders from the xdg dirs when you run it which sucks but im to lazy to create a button for it.
- currently supports ALPM (arch btw)

## stuff that could potentionally maybe be added at some point
- support for more package managers (dkpg, the fedora one and more)
- better frontend (GTK4?)
- actions for uninstalling packages and removing folders directly in the app
- showing diff from last run
- ability to mark packages and folders so they get hidden
